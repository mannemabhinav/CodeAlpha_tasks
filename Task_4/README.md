# Sales Prediction using Python

## Overview

This project predicts product sales using machine learning based on advertising spending across different marketing channels.

---

## Objective

- Predict future sales based on advertising expenditure.
- Analyze the impact of advertising channels on sales.
- Build a regression model for sales forecasting.
- Generate actionable business insights.

---

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## Project Workflow

### Data Collection

- TV advertising budget
- Radio advertising budget
- Newspaper advertising budget
- Sales data

### Data Preprocessing

- Loaded the dataset
- Checked missing values
- Removed unnecessary columns
- Prepared features for training

### Model Building

- Applied Linear Regression
- Split data into training and testing sets
- Trained the model
- Generated predictions
